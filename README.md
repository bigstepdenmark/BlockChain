# BlockChain

### Getting started

1. Clone or just download the repository.
```bash
➜ git clone https://github.com/bigstepdenmark/BlockChain.git
```

2. Build the project
```bash
➜ mvn package
```
<img src="images/1.png">

3. Run the generated .jar file
```bash
➜ java -jar target/BlockChainSI.one-jar.jar
```
